# To-Do List 
## Link:
https://noah-gullo.github.io/todo-list/

## Description:
A complete To-Do project that allows the creation of tasks under different projects. Each project has its own list of tasks. Users can create/delete projects and tasks. All data is saved with the option to reset your data at any time.

## Skills:
This project was good practice at saving data to localStorage along with further JavaScript and HTML/CSS practice on a project of a larger scale. Furthermore, it was good practice using bundlers and seperating the DOM/application logic. It also improved my debugging skill by working through bugs related to complex application logic.

## Reflection: 
This project definitely took significant ammounts of planning and really emphasized the importance of ensuring that I have a clear path forward before writing any code. Furthermore, I tried to add little bits of user responsiveness as the user iteracts with the page and considered a bit of UI/UX.
